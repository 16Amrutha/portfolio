# 🌌 Amrutha's Portfolio

A modern, interactive portfolio website built using **React**, **TanStack Start**, **TypeScript**, and **Tailwind CSS**.

This portfolio showcases my projects, research, achievements, technical skills, certifications, and experience in Artificial Intelligence, Data Science, and Software Development.

---

## ✨ Features

- Modern animated UI
- Interactive cosmic background
- Responsive design
- Glassmorphism effects
- Smooth scrolling animations
- Project showcase
- Skills section
- Research publications
- Certifications
- Contact section
- Fast routing with TanStack Router
- Type-safe development using TypeScript

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

Move into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will run at

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
